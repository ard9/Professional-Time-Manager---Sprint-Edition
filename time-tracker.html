💾 Data Storage & Transfer Guide

🔍 Where is data stored?

All your data (tasks, sprints, statistics) is stored in your browser's Local Storage. This means:

  • ✅ Data exists only on this computer and this browser
  • ❌ If you clear browser data, everything will be deleted
  • ❌ Not accessible on other computers or browsers
  • ❌ Not available on your mobile or tablet

📤 How to transfer data?

To transfer data to another system or browser:

  • 1. Click the "Export Data" button
  • 2. A JSON file will be downloaded
  • 3. Transfer this file to your other device
  • 4. On the new device, click "Import Data"
  • 5. Select the JSON file

💡 Recommendations

  • 🔄 Backup your data regularly
  • ☁️ Keep the exported file in Google Drive or Dropbox
  • 📧 Email the file to yourself for safekeeping
  • 🗓️ Create a weekly backup
